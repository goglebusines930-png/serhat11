# mini-gpt

## Kurulum Talimatları

### 1. Python yüklü olsun (3.10+ önerilir)

### 2. Klasör oluştur ve içine gir
```bash
mkdir mini-gpt
cd mini-gpt
```

### 3. Gerekli kütüphaneleri kur
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cpu
pip install numpy tqdm requests
```
